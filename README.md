# デマンド分析ツール（Streamlit / ローカル）

30分値データ（xlsx / csv）をアップロードし、施設単位でデマンド分析を行うローカルWebアプリです。  
既存MVP機能を維持したまま、Excel分析シート相当の表示を拡張しています。

## 動作環境
- Python 3.11+
- ローカル実行のみ

## セットアップ
```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

## 起動
```bash
streamlit run app.py
```

## 入力仕様
- 形式: `xlsx` または `csv`
- `xlsx`: シート選択（施設）
- `csv`: 施設名入力（必須）
- 入力単位:
  - `kW`（30分平均電力）
  - `kWh`（30分エネルギー）→ 内部で `kW = kWh × 2`
- 内部計算単位は常に `kW`

## 日中オプション仕様
- デフォルトは **全時間（0:00-24:00）集計**
- `日中時間帯オプションを有効化` を ON にした場合のみ:
  - 日中時間帯スライダーを表示（初期値 `9:00-18:00`）
  - 日中関連の指標・表・グラフを計算/表示
- OFF 時は日中スライダー非表示、日中関連は未計算

## 時刻正規化
入力が区間終了ラベル（例 `00:30` が `00:00-00:30`）でも、内部では以下へ正規化します。
- `00:00, 00:30, 01:00, ... , 23:30`

対応例:
- `00:30` / `0:30` / `00:30:00` / `0時30分`

## 実装済み分析
### 既存MVP（維持）
- サマリー
- 日別ピーク推移
- 平均負荷曲線（48スロット）
- 曜日別平均負荷曲線
- ヒストグラム
- 日中時間帯分析

### 拡張機能
- グラフ（平均値）
  - 年間デマンド平均
  - 月別デマンド平均（1〜12月選択）
- グラフ（月別実績）
  - 月選択時に `日別 × 48スロット` の実績曲線を複数系列で表示
- 月別標準偏差
  - 30分版 / 1時間版
  - 全時間 / 任意時間帯（標準偏差セクション内で設定）
- 曜日別標準偏差
  - 30分版 / 1時間版
  - 全時間 / 任意時間帯（標準偏差セクション内で設定）
  - 1時間版は「30分データを1時間kW平均へ再集計した系列」で標準偏差を算出（30分標準偏差の平均ではない）
  - 標準偏差は Excel と同じ `STDEV.P`（`ddof=0`）固定
  - セル数値の文字色切替（黒 / 濃灰 / 自動コントラスト）
- ヒートマップ配色
  - 3色（青→白→赤）
  - 各テーブルごとに `min / median / max` を自動算出
- グラフ（月別日中デマンド）
  - 月別で `最高/最低/平均デマンド(kW)`、`全時間使用量(kWh)`、`日中使用量(kWh)`、`日中利用率(%)` を表示
  - 全時間使用量と日中使用量をグラフ表示
- 時間別デマンド回数等
  - `日最大デマンドを記録した時刻` の回数（棒）
  - その時刻での最高デマンド値（折れ線）
  - ツールチップは日本語
- デマンドレンジ
  - レンジ幅可変、回数・割合表示
- ピークカット目安試算
  - 入力UI + 試算結果表示（参考・簡易試算）
- 入力データ可視化（色付き表）
  - 行=日付、列=30分時刻のヒートマップ
  - 月フィルタ / 期間フィルタ / 表示行数上限対応
- (月毎データ) / (日)〜(土) 相当表示
  - フィルタ式で段階表示（大量一括表示なし）

## ダウンロード
- 主要グラフごとに PNG ダウンロード
- 各表示データの CSV ダウンロード
- 標準偏差テーブル: CSV + PNG ダウンロード
- 入力データ色付き表: CSV + PNG ダウンロード
- PNGは `kaleido` が使えない場合でも `matplotlib / Pillow` フォールバックで生成

## パフォーマンス
- `st.cache_data`: シート一覧取得・データ読込をキャッシュ
- `session_state`: 分析結果を保持し再計算を抑制
- 重い計算は `分析実行` ボタン押下時のみ

## ファイル構成
- `app.py`: アプリ本体
- `demand_parser.py`: ファイル読込・時刻正規化・単位変換
- `analysis.py`: 集計・分析ロジック
- `ui_sections.py`: 描画・ダウンロードUI
- `requirements.txt`: 依存ライブラリ
- `README.md`: 本ドキュメント

## 実データ確認
ワークスペース内の以下で動作確認しています。
- `デマンド分析ツール.xlsx`（シート定義確認）
- `施設デマンドデータ一覧.xlsx`

`デマンド分析ツール.xlsx` の `デマンド` シートはテンプレート用途で30分実績値が空の場合があります。  
その場合はアプリで「有効な30分データが見つかりません」と表示されます。
